题目描述

给定一个二叉树，判断它是否是高度平衡的二叉树。

本题中，一棵高度平衡二叉树定义为：

一个二叉树每个节点 的左右两个子树的高度差的绝对值不超过1。

示例 1:

给定二叉树 [3,9,20,null,null,15,7]

返回 true 。

示例 2:

给定二叉树 [1,2,2,3,3,null,null,4,4]

返回 false 。

解题思路

如果一个二叉树是高度平衡二叉树必须满足以下几点

  • 左右根节点的左侧支线和右侧支线高度差绝对值小于等于1
  • 先判断左右支线高度差，如果绝对值不超过1，将左右支线当做根节点递归使用该方式判断
