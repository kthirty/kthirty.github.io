题目描述

给定一个链表，判断链表中是否有环。

为了表示给定链表中的环，我们使用整数 pos 来表示链表尾连接到链表中的位置（索引从 0 开始）。 如果 pos 是 -1，则在该链表中没有环。

首先需要知道环形链表是什么？
环形链表：链表中的某一项的下一项指向链表该项的前面某个节点，形成一个闭环的链表

总结

使用双指针，快指针每次走两步，慢指针每次一步，如果不是环形链表，快指针会先走到终点，否则，如果快指针不是null说明慢指针也不可能为null，无论环形有多大，在慢指针与快指针总是会相遇，相遇说明是慢·指针
