微服务第一章之Consul UI安装

Consul是一个用golang开发的分布式高可用服务治理与与服务配置的工具，它有以下功能：

1. 服务发现：

Consul客户端可以提供服务，其他客户端可以使用服务名称通过DNS以及http等协议来发现服务，有助于提高服务的可扩展性。

2. 健康检查：

Consul的客户端可以通过多种方式来检查某个服务或当前节点是否处于健康状态，比如查看一个服务是否可以返回200 OK，或者是查看client当前部署的机器的内存使用是否在80%以下。

3. key/value存储：

Consul客户端可以使用http api来存储、修改、删除一些k/v键值对。

4. 安装

下载Zip后将consul.exe所在目录路径放入Path变量中
[Consul下载]: https://www.consul.io/downloads.html

5. 运行

cmd中执行 basic consul agent -dev 运行，在浏览器访问 [http://localhost:8500]:http://localhost:8500 可以看到可视化界面
