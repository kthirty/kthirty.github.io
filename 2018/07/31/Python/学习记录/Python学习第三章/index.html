3. 函数的参数

  • 位置参数 : def func(a,b) a,b参数为位置参数，即必传参数

  • 默认参数 : def func(a,b,c=0) c参数为默认参数，该参数可传可不传
    注意，默认参数不要使用指向性的，例：

    该函数是有问题的，原因是，方法在定义的时候，会自动创建一个默认参数所用的变量，如果此参数是有指向性的可变参数，且在函数中改变了该参数，那么这个变量也会改变，下次再调用时，这个参数的就不再是[]了，而是上次操作之后的数据

  • 可变参数 def func(a,b,*args) *args为可变参数，可以接受0-无数个参数
  • 关键字参数 def func(a,b,*,d,e) d,e为关键字函数
    1. 关键字参数可以不传或者全传，但是不能只传递其中一部分
    2. 关键字参数也可以是默认参数，如果是默认参数，可以不传
    3. 关键字参数传入示例: func(1,2,d=3,e=4)
  • 命名关键字参数 def func(a,b,**kw)
    1. 命名关键字参数接受自定义命名的参数
    2. 参数名不受限制，系统用dict接受
    3. 传入示例： func(1,2,demo='3')
  • 参数传递
    1. param的传递，*+list or tuple 可以依次传入参数
    2. ** + dict 可以依次传入命名关键字参数 or 关键字参数

4. 递归函数


尾递归函数
1.返回值不能包含表达式，如果参与计算，计算机会记录每次调用
2.如果语言对尾递归函数进行了优化，可以解决栈溢出问题
