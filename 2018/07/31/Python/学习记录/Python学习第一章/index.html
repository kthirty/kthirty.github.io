Python学习记录第一章--安装篇

1.下载安装包

首先，根据你的Windows版本（64位还是32位）从Python的官方网站下载Python 3.7对应的
64位安装程序或32位安装程序（没梯子请移步国内镜像），然后，运行下载的EXE安装包：

箭头标志表示将Python加入Path变量中，无需手动配置

2.运行Python

打开Cmd，输入Python，出现以下界面表示安装成功

可能出现错误

出现原因为安装时为勾选Add Python 3.7 to PATH
将python.exe加入到Path变量中可以解决

至此，安装已完毕
