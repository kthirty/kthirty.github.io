GitHub个人博客搭建

发表于 2018-08-29 | 更新于 2019-01-20 | 分类于 技术 | 评论数：
使用hexo搭建自己的博客，并托管至GitHub,使用到的技术hexo、git、markdown

1 准备工作

  • GitHub
    • 账号
    • 客户端Git for Windows
  • Nodejs
  • Hexo

2 创建你的博客

2.1进入你希望创建的目录

2.2初始化博客

打开浏览器验证是否已启动 Open

2.3使用IDE打开你的博客项目

  1. 推荐使用HBulider
  2. 打开_config.yml配置文件
  3. 修改deploy

3 部署到GitHub

3.1 准备部署

  1. cmd进入项目根目录
  2. hexo g 生成部署文件

  3. 配置GitHub

    1. 打开GitHub new repository 创建一个你的项目，项目名yourname.github.io
    2. 打开Setting -> GitHub Pages 开启你的pages
    3. 配置本地Git

  4. hexo d 部署至GitHub

    此处可能会出现 error deployer not found:git
    原因为缺少hexo-deployer-git插件
    执行npm install hexo-deployer-git --save可以解决问题

遇到的问题

  • yml文件配置 name: value value和:之间必须包含一个空格
  • 简要信息过短 可以使用 <!-- more --> 占位

至此项目已经部署完毕，打开yourname.github.io查看你的博客吧
