Centos7-Mysql5

发表于2020-04-09|更新于2025-10-14|技术环境搭建MySQL
|字数总计:485|阅读时长:2分钟|阅读量:

Centos7下已tar包形式安装MySQL5.6

删除Centos7 自带的Mariadb（如果没有直接跳过）

rpm -qa|grep mariadb
rpm -e --nodeps mariadb-libs-5.5.44-2.el7.centos.x86_64

#删除etc目录下的my.cnf文件

rm /etc/my.cnf

#检查mysql是否存在
rpm -qa | grep mysql

#检查mysql组和用户是否存在，如无创建
cat /etc/group | grep mysql
cat /etc/passwd | grep mysql

创建mysql 组

groupadd mysql
#创建一个用户名为mysql的用户并加入mysql用户组
useradd -g mysql mysql

下载mysql安装包

wget https://mirrors.tuna.tsinghua.edu.cn/mysql/downloads/MySQL-5.6/

解压并修改名称（安装到/usr/local/目录下）

tar -zxvf mysql-5.6.46-linux-glibc2.12-x86_64.tar.gz
mv mysql-5.6.46-linux-glibc2.12-x86_64 mysql

创建data目录（如果有就不用了）

cd /usr/local/mysql
mkdir data

更改文件权限到mysql:mysql 下

chown -R mysql:mysql /usr/local/mysql

在/etc下创建my.cnf 文件(内容如下)

安装并初始化

cd /usr/local/mysql
./scripts/mysql_install_db --user=mysql --basedir=/usr/local/mysql --datadir=/usr/local/mysql/data
cp ./support-files/mysql.server /etc/init.d/mysqld
chown 777 /etc/my.cnf
chmod +x /etc/init.d/mysqld

启动mysql

/etc/init.d/mysqld restart

修改密码（在mysql命令行执行）

set PASSWORD = PASSWORD(‘111111’);
flush privileges;

添加远程访问权限（Mysql命令行执行）

use mysql
update user set host=’%’ where user=‘root’;
/bin/systemctl restart mysql.service

将mysql添加到bin中，方便使用

ln -s /usr/local/mysql/bin/mysql /usr/bin/mysql

备注

安装方式没有初始密码，mysql -u root -p 不需要输入密码，直接回车
